## 2750. 수 정렬하기
#### 알고리즘: Bubble sort
#### 시간 복잡도 계산
    시간복잡도: O(N^2)
    한 바퀴 도는 데 O(N) X 최악의 경우 (N-1)바퀴
    한 바퀴 돌 때마다 원소 1개씩 제자리로 간다. -> 어차피 (N-1)번 돌면 모든 원소가 제자리를 찾게 됨.
    이미 어느 정도 정렬되어 있다면, 바퀴 수가 줄 수 있도록 개선 가능함.
#### 아이디어
    두 원소의 자리를 바꾸고 싶다?
    ```
    if a[i] > a[i+1]:
        tmp = a[i]
        a[i] = a[i+1]
        a[i+1] = tmp
    ```
